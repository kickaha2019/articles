	I help my and Virginia's soft toys to do things they couldn't otherwise do, like speak.  Move around.  And make games.  I felt the game they wanted to make was about the end of the world, and how to make sure it's enjoyable.

	Hence [https://www.alofmethbin.com/Games/Armageddon/release/ Armageddon] was written using Adobe Flex and Flash, and released at the end of 2011.  The game has you answering a set of questions.  Depending on the answers you get a different disaster.  There is a set of answers which does not result in a global apocalypse.
